前言

先这篇文章的目的单纯是为了更加简洁，更加方便查找。另外本文的编写会在求积分，求极限，求导数教学的基础上进行。另外声明我不是原创（原创是我们的超哥）大家可以自行搜索三大计算即可知其大名。废话少说，直接开始正文

求极限

首先我们拿到一个求极限的题，第一步当然是判断极限类型。经过总结我们可以把它分成七种未定型。

0/0型

有以下处理方法：

1.用泰勒展开

所谓遇事不觉，选泰勒。用泰勒解题，其实就是把分式化成一推x的a次方的集合来求解，因为根据公式所有非x的方前面100%是一个系数。话都到这个份上了，想必各位用脚想都知道，在0/0求极限中，这个方法有多吊，基本上就是无脑展开，然后加减乘除。
想用好泰勒公式需要以下几个非常重要的条件
首先需要对以下8个公式非常的熟练,包括变型。

其次需要明白泰勒的展开原则
原则一（泰勒唯一的难点）：
上下同阶即分子分母的阶数相同，这里需要很强的观察力，在泰勒单式展开的表达式又需要开平方或立方时需要把同阶的项都写出来，最好在展开时直接就展开到公式的同分母同阶再代入变量，直接不讲道理。

原则一补充

写这里主要是为了帮助加深对原则一的理解，各位知道为什么等价无穷小为什么可以在乘法中用而不能在加法中用吗？其实加法也可以用，但要分情况。我们先来说说乘法，在0/0中，其实决定最后的极限值的是最小的那个幂，也就是说在0/0中决定极限值的是全式子最底阶的那个元素，（相反在无穷比无穷时则时最高阶的决定论，但是那里一般不用泰勒展开。我们就暂时不讨论）而无穷小的展开恰好是只展开到了最低阶，因此在乘除法中我们只需要关注最低阶就完全够了，但是加减法不一样啊，加减法它最低阶级可能会抵消掉的这也是为什么有第二原则的原因，因为抵消不了的最低阶才有价值。

原则二（勇敢计算的来源）：
用泰勒公式求未知数时，分子需要相互展开到彼此抵消不了
当你符合了以上两点要求时，多做题找感觉，泰勒公式这一块应该就没问题了。

2 洛必达求导

想用好洛必达求极限必然要知道这个理论的使用场景，这里笔者总结了以下几种可以使用的场景和不能使用的场景。
可以使用的场景：
1：无穷比无穷，或者0/0。
2：分子分母要在这个点的区间可导。
3：洛必达后的分式极限要存在。注意这里的因果关系，是因为极限存在才能推出洛必达相等。
不可以使用场景：
1：当分子或者分母中有1/cosx，或者1/sinx。不可以用洛必达法则。

洛必达求极限补充

1：洛必达求极限往往一般和无穷小替换结合使用。
2：求导前尽可能先化解

洛必达补充

1：对于极限后因为有界量导致极限不存在而不能用洛必达求导。我们可以用无穷小乘有界量来处理掉有界量。
2：对于在无穷小乘无穷大中在转化的的过程中求导复杂的往往放在分子求导。
3：再无穷减无穷时的三种处理方式。
1）通分
2）有理化
3）倒带换

3 等价无穷小

等价无穷小其实就是泰勒公式的简化版。但是我们需要搞清楚等价无穷小替换的原理，来防止我们错误使用，和加快我们的解题速度。其实等价无穷小的替换使用和上面泰勒的使用是一样的。但是为了便于使用，推荐还是只在乘除法中用（有些分式可以去加减号，比如对数函数的运算）。比如在无穷乘等价无穷小时，等价无穷小就可以直接替换。此外我们在了解这个原理的基础上，我们还可以可以通过构造等价无穷小来加快解题速度。

4 导数定义

用导数的定义求极限
首先一定要搞清楚导数定义的表达式。（分母是分子自变量的差值，结果时相同部分点的导数值。）
然后要学会构造函数向导数定义上靠，（在构造的过程中要注意自变量的趋向）
下面是读者总结的一些适用场景（这个很重要）：
1：用导数定义求极限往往时0/0的最后处理方式也就是说洛必达和泰勒都不能用时可以考虑构造导数定义来求极限
2：题中出现“函数在某一个点导数存在”这样的字眼。一般会用到导数定义，切记如果在0/0中一定不能用洛必达。因为洛必达要求函数在这个点的去心临域都可导，所以用导数定义，用一阶导的形式表达二阶导在零点的值。
3：如果说f(x)有二阶连续导数，那可以对f(x)求两次导，如果说f(x)的二阶导数存在就只能求一次导。因为导数存在从来都不代表导数连续，可导。

求极限的运用

1.连续与间断点

在求间断点过程中最重要的是间断点的来源。如何通过求极限的知识求解。
间断点的来源主要两个方向：
1：分母为零的点
2：函数没有意义的点
可能有读者会问这不是一样的吗？其实不然，把这两类来源分开讨论在找间断点的过程会更加清晰，分类更加准确，这很重要。
这一部分可以加强自己的分类讨论能力。

2.求渐近线

求渐近线主要三种，主要是斜渐近线首先基本公式要练习好，然后可以去进阶快速解法，相信我，快捷解法的速度会让你觉得不真实，再这之前一定要搞清楚快捷解法的原理用原理。知原理才能巧妙的解题。
在求渐近线的过程中会综合很多我们前面用到的求极限小技巧，知识等等。一定把前面的基础打好，在来练习这里时自然水到渠成。

求极限补充

知识点

1：e^x,arctan(x),[x],根号下的X方（这个也要区分x趋于无穷时正负趋向时不同），分段函数的左右极限不同，这一点往往是得分的关键
2：知识点 第一个重要极限sin(x)/x=1,无穷小乘有界量为0

高阶函数和低阶函数

这一部分时自己关于求极限的思考，所谓极限极限，不就时一个趋向某一个值的快慢问题吗？一想到速度不就是导数吗，导数的定义不就是速度的定义吗，而泰勒的先决条件也是函数有n+1的导数存在，洛必达则更为直接，直接就是求导，强行比大小。某一点的极限的话就直接代进去用导数的大小表示速度的大小，然后用速度的比值来推算在这个点距离的比值关系，来确立极限。这样来说的话，某个点的高阶函数是不是可以认为比这个点的低阶函数更快的接近这个值，所以高阶无穷小比低阶无穷小更快的趋向于0，可以这么认为当有一天高阶无穷小到了0的时候，低阶肯定没到。所以高阶无穷小比低阶无穷小等于0.同理我们这样定义一个函数高阶无穷大。比如x->00 ln(1+x)/x，求导或者直接从图像上可以看出，x趋向于00的速度大于ln(1+x)所以我们这么认为x是ln(1+x)的高阶无穷大，当有一天x到一个很大的数的时候，ln(1+x)一定比它小，所以低阶无穷大除以高阶无穷大等于0。这是未定型的计算，自然有定型计算，那个如果在那一点有定义的话，完全可以直接代进去。没定义我们就得算这个点的极限是多少总，这里就引申出了连续和间断点的问题，当函数在这个点的极限等于这个的点函数值不就时连续嘛。我感觉在比值上极限就是个速度问题，这里真正的方法也只有泰勒和洛必达。别的都只是往上面靠。而有时候极限也可以表示距离无限靠进，这就有了渐近线就时函数和渐近线无限靠近，这才有了快捷解法。
以上就是我对函数求极限自己的一些理解，没有任何参考资料，单纯就我的经验和思考。嘻嘻。

求导数

首先要记得一句话，记得求导之前先化简，比如lnf(x)/g(x)=lnf(x)-lng(x)，要不然容易算自闭。

利用导数定义求导

这个方法主要用于以下4给场景（标志一般为一个函数再某一个点可导，求函数在某一点的函数值）
这种求导一般是求某一个值
1：被求导的函数太过于复杂（重点）（比如很多项在一起做乘法）
2：被求导函数含有抽象函数
3：被求导函数不符合求导法则
4：分段函数分段点求导用导数定义求导（重点中的重点）
补充，这是一种方法虽然用法固定，但是是一种方法，有时候可以从这个角度去思考。

复合函数求导

复合函数求导主要是为了后面的积分换元做准备，所以再练习的时候要注意反向推导。
要做好复合函数求导需要熟练掌握以下两点要求：
1：求导法则
2：求导公式
这里再分享一个心得:优先大函数，从外到内层层递进。可以括号这种标志符。
再复合函数求导的过程中一定要注意计算方法，不要心急，一步一步求导从外到内。

隐函数求导

这里一般有两种情况：
1：求导函数直接等式两边对x求导，把y看成x的函数
2：求dy,先化简，这种题一般要先化简，然后等式两边同时微分，注意是同时微分。

参数方程求导

参数方程求导的过程种一定要搞清楚是对谁求导，以及最后结果的原理是除dt乘dt这样的结构，而二阶导注意是一阶导的结果再对t求导，不是y再对t求导。然后加强计算能力就可以了（相信我有些求导，导着导者就会发现有一个复合函数导错了），参数求导只要把求导的原理搞清楚了，一般就没什么问题。即使是有一些隐含数也一样。
这里笔者有一个小tip:可以在每次的结果后面加上是谁对谁求导比如（dy/dx）。这样对于一些复杂题的处理会更加清晰。

分段函数求导

分段区间正常求导，分段点用导数的定义求导（这个很重要，且分段点导数再这个点是一个值不是一个公式注意）。记住公式不要错，思路要清晰。

变限积分求导

这是一个非常重要的求导过程，变限积分求导要记住一个很重要的原则，X只允许出现在积分的上下限里。
1.如果在函数外就提出来,用链式求导法则求导即可。
2.在函数的自变量替换再整体对U求导（这里一定要注意积分上下限也要替换）。
在保证了上面这个原则后，就可以用变限积分求导的公式（这里可以巧记把上下限代入自变量然后相减再整体求导）。

凑微分法求积分

既然都写到这里了我就不得不点名批评一组积分公式，其行为简直不要太恶劣。

闲话少说让我们直接开始正文
用凑微分法求积分，要求读者对一些常见的求导过程非常熟悉，主要是复合求导，因为凑微分法求积分其原理到底其实就是凑微分法的逆向过程。凑微分的方式多变，但是我们其实只需要找到中间变量及中间变量的导数，然后稍加处理问题自然就会迎刃而解。
基于这一原则，我们可以得出其实看似复杂的求积分其难度应该更小，因为有求导法则的限制，复杂的函数往往比较容易找到其实规律，相反看似形式简单的题目比如f(x)=根号tanx这种函数。但是不要慌虽然变化多，但也不是没有规律。以下是笔者总结的几种常见的变形，及处理方法

三角代换

这一类变化，对于处理1/2，3/2，4这种阴间幂数。往往有奇效，因为三角代换可以把式子的幂从1变成2，把两项变成一项。注意不要忘了dx要化为等效的dt。

根式代换

这主要是对于不好处理的根号比如上文中的f(x)，这样可以消去根号，把式子变成有理函数积分来处理，这里读者可以参考我的有理函数通解一文。

三角变化里的三角代换

主要有，和差化积，积化和差，升幂，降幂，sinx cosx的转化，提取(cscx)^2（secx）^2来凑tanx,cotx。凑（sinx+cosx）和
（sinx-cosx）的导数。凑分子为导数的结果。

e^x的一些变化

这种除了公式外，主打的就是一个e^x求导等于e^x抓住这个一般没问题，一般是公式，和加一减一的变化，也有一些根式代换
总之在凑微分求积分的过程中变化很灵活，比如笔者上面所写的几种方法也很有可能一起考，这在凑微分求积分中可以说非常常见。读者一定要多多练习，做到熟能生巧。当你足够熟练的时候或许就会发现凑微分并没有想像中那么难。

分部积分求积分

分部积分法，作为链式求导的另一种形式，可以帮我们用另一种角度来转化求积分的式子，比如像arctanx，lnx,ln(x+根号1+x^2)这种的不好凑微分的时候，分部积分法可以帮助我们先对这个式子求导然后再来求积分。有时候甚至可以强行分部积分（先对要分部的部分求积分求出原函数后再凑进去），在分部积分法中我们常常用到两大公式。

表格法

主要是针对幂函数的消幂功能。适用范围，X^a x sinbx,X^a x cosbx,X^a x e^bx,X^a x lnbx。
口诀是对幂函数依次求导，对三角函数或者指数函数依次求积分，相互对应，然后正负相间，斜向相加。

分部积分法补充

在练习分部积分法时，要时刻记住这只是一种方式，有许多题其实可以用别的方法，甚至很有可能比分部积分法更快。另外分部积分法是使用有时候也可以对分母凑微分，甚至可以抵消。有很多妙用。多多练习必有收获。

有理函数积分

这里的求积分教学可以参考我的“有理函数求积分通解”。里面有较为详细的讲解。这里补充两个小技巧
1：对分母同乘变量，提取变量的操作
2：分离分母的操作，一般就是分子的加减变化
总之：在处理很多积分式的时候不可只想着怎么去搞分子，也要加强对分母的处理能力。

三角函数积分

三角函数积分是难度区间有点大，这里笔者主要分享几种常用到的处理方法。
1：化分母为方，然后提取分母凑微分一般就是sinx方和cosx方
2：万能公式即令U=tan(x/2)可以把三角函数积分化为有理函数积分处理，这里的sinx,cosx,du。读者可自行推导
3：反三角代换
4：变1=sinx方+cosx方
5：正常凑微分，或者分部积分，即可
